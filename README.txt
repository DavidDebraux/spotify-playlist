Spotify-Playlist

Spotify-Playlist est un outil web qui se connecte à un compte spotify existant,
prends un fichier excel ou csv et récupère une liste de titres avec le nom de l'artiste et l'ajoute à une nouvelle playlist.

MVP

1. une page de connexion au compte Spotify-Playlist.

2. une fois connecté, une nouvelle page pour permettre l'uplaod (drag&drop) d'un fichier .xlsx ou .csv et ajout d'un libellé pour la playlist à créer.

3. lecture ligne à ligne pour ajout du titre de l'artiste dans la nouvelle playlist péalablement créée.

Après

1. liste des playlist existante pour ajouter les titres trouvés à une playlist deja existante
