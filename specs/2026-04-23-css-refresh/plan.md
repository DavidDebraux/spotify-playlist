# Plan: Refresh CSS et apparence générale

## Objectif
Moderniser le design de l'application tout en gardant la fonctionnalité.

## Principes
- Design épuré et minimaliste
- Boutons plus compacts sur desktop (comme demandépreviously)
- Layout vertical sur mobile, possiblementisé sur desktop si nécessaire
- Couleurs: garder le vert Spotify (#1DB954) comme accent

## Étapes proposes
1.audit du CSS actuel
2.définir une palette de couleurs cohérente
3.redessiner les composants principaux:
   - Boutons (Spotify green, taille adaptive)
   - Cards et zones (bordures, shadows)
   - Formulaires (inputs, selects)
   - Mode switch (Fichier/Image)
4.tests responsive sur mobile et desktop

## Contraintes
- Garder la compatibilité mobile
- Ne pas casser les fonctionnalités existantes
- Demander confirmationavant chaque modification significative
