# Mission

Spotify-Playlist est un outil web qui permet de créer des playlists Spotify à partir de fichiers Excel/CSV.

## Objectif

Permettre aux utilisateurs de:
- Se connecter via leur compte Spotify existant
- Importer un fichier (drag&drop) contenant des titres avec nom de l'artiste
- Créer automatiquement une nouvelle playlist avec ces titres

## MVP

1. Page de connexion OAuth Spotify
2. Page d'upload de fichier .xlsx ou .csv avec libellé de playlist
3. Lecture ligne à ligne et ajout des titres à la playlist créée

## Phase future

- Gestion des playlists existantes pour ajout à une playlist déjà créée
